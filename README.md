# teaching-comic

專案用途：製作教學用的漫畫

## 專案簡介
本專案旨在運用 AI 與各式工具，生成與編輯適合用於教學情境的漫畫。

## 目錄結構
- `ANTIGRAVITY.md`：專案規範與 AI 工作守則。
- `skills/`：專案專屬 AI 技能目錄。
  - `skills/comic-generator/SKILL.md`：教材四格漫畫生成技能。

## 開發與設定
本專案支援**教材四格漫畫生成技能 (comic-generator)**：
   - 當您提供教材檔案路徑或直接貼上教材內容時，AI 會自動為您整理重點，並生成對應的 2x2 四格漫畫圖片，輸出至 `output/` 資料夾下。
   - 支援日系黑白、美式英雄、Q版卡通、皮克斯3D、復古手繪、卡皮巴拉水豚風等多種風格。
